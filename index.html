Git cheatsheet

Ta strona jest dodatkiem do mojej prezentacji o podstawach gita, dostępnej tutaj

1. Konfiguracja Gita

Konfiguracja tożsamości

$ git config (--system/--global/--local) user.name "nazwa uzytkownika"

$ git config (--system/--global/--local) user.email example@example.com

Konfiguracja edytora

$ git config (--system/--global/--local) core.editor (vim/nano/inny...)

Konfiguracja konfiguracja narzędzia rozwiązywania konfliktów

$ git config (--system/--global/--local) merge.tool (vim/nano/code/inny...)

2. Repozytorium lokalne

Tworzenie repozytorium

$ git init

Dodanie plików do repozytorium

$ git add <pliki (może być wyrażenie regularne)>

Tworzenie nowego brancha i przełączenie sie na niego

$ git checkout -b <nazwa brancha>

Zmiana aktywnego brancha

$ git checkout <nazwa brancha>

Commitowanie zmian (uruchamia domyślny edytor do napisania treści commita)

$ git commit

lub

$ git commit -m "Treść commita"

Mergowanie zmian (będąc na branchu docelowym)

$ git merge <nazwa mergowanego brancha>

3. Repozytorium zdalne

Klonowanie zdalnego repozytorium

$ git clone <link do repozytorium>

Dodawanie repozytorium do listy zdalnych

$ git remote add <lokalna nazwa repozytorium> <link ssh lub http>

Listowanie zapamiętanych zdalnych repozytoriów

$ git remote -v

Pobieranie zmian ze zdalnego repozytorium

$ git pull <nazwa zdalnego repozytorium>

Wypychanie zmian do zdalnego repozytorium

$ git push <nazwa zdalnego repozytorium> <branch>
